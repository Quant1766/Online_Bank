# Online Bank

### API

| Route              | Method | Purpose                                 |
| ------------------ |------  | --------------------------------------- |
| /api/accounts      | GET    | Get all users                           |
| /api/accounts/id   | GET    | Get single user with ID                 |
| /api/accounts      | POST   | Post new user. Fields: "name": str      |

### Functionalities
* Load money with route: /load/[account_id]
